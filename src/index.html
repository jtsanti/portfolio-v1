Cosmic Tic Tac Toe

Main Skills
Angular 15.2, TypeScript, SASS, Nebular 9.0
Backstory

As a UI/UX Developer with United Systems & Software, I was learning Angular on the job and a lot of what I was doing was working with their stylesheets in SASS and modifying existing components. While I learned a lot from that experience, I wanted to whip up something to give me more exposure to building an Angular app from the ground-up and to play with a bit of the functionality.

Making a Tic Tac Toe game seemed like a straightforward way to experiment with passing values between components, along with finding an excuse to build a custom (albeit simple) pipe. Since I was trying new things, I also came across the Nebular UI kit and added that to the project, which came pre-loaded with some nice visuals to make theming fairly quick.

Conference Now Websites

Main Skills
HTML5, CSS3, Bootstrap 4+, JavaScript
Backstory

As a Product Analyst with Web Courseworks, our LMS (Learning Management System) was build upon Moodle's framework (PHP, JSON, AJAX) and we retooled this product during the pandemic to produce virtual conference for our clients. This involved working with stakeholders to scope out previous in-person conferences the clients held, and then converting that vision into an online user experience that could rival what their members were familiar with.

A large part of this role involved wireframing and designing the UI from the ground up, and then building out each page with HTML, CSS, and JavaScript.

Find a Film

Main Skills
HTML5, JavaScript
Backstory

While looking through Codecademy's frontend engineer program, I wanted to get some practice in with RESTful APIs so I created this quick app to pull movie data from The Movie Database and then display it to a user. The app has two modes available:

  • Search for a movie and, if there's a hit, show the movie cover and some info.
  • Choose a genre and a random movie from that genre will display. You can then like or dislike the movie to show a different random movie, and a tally of your choices will be displayed on the page.

Etch-a-Sketch

Main Skills
JavaScript
Backstory

I wanted to do a quick project to play around with DOM events and manipulation, so I created this Etch-a-Sketch game. It uses JavaScript to generate the game grid and reset it when needed, along with adding dynamic styles while the user is playing.

I may come back to this in the future and try to figure out a way to make this work with key events instead. I think that would make the game more challenging, along with making it more reminiscent of the real-life game which was pretty hard (in my opinion). The UI is also fairly basic at the moment, and although this project was mainly geared towards functionality, I might also modernize this at some point to make it prettier.
